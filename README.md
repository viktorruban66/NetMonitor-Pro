📊 NetMonitor Pro — монитор сети в реальном времени
Профессиональный монитор сетевого трафика с графиками скорости, статистикой и оповещениями.
Отображает скорость загрузки и выгрузки, общий трафик, пиковые значения и историю.
Реализован на 7 языках программирования для демонстрации работы с сетевыми интерфейсами и визуализацией данных.

🧠 Концепция
NetMonitor Pro — это мощный инструмент для наблюдения за сетевым трафиком в реальном времени:

✅ Скорость загрузки (download) и выгрузки (upload) в Мбит/с или Кбит/с.

✅ График скорости — отображение изменений за последние 60 секунд (или минут).

✅ Общий объём переданных данных (входящий/исходящий).

✅ Пиковая скорость и средняя скорость за сессию.

✅ Выбор сетевого интерфейса (в GUI-версиях или через команду).

✅ Уведомления при превышении заданного порога скорости.

✅ Автообновление с настраиваемым интервалом.

✅ Сохранение истории в CSV/JSON для последующего анализа.

✅ Красивый интерфейс (в GUI) или цветной ASCII-график (в консоли).

🚀 Как запустить
Для работы с реальными сетевыми интерфейсами потребуются системные библиотеки.
В консольных версиях используется симуляция для демонстрации (можно адаптировать под реальные данные).

bash
# Python (GUI + Matplotlib)
pip install matplotlib psutil
python netmonitor_python.py

# C++ (Qt + psutil-like библиотека)
qmake && make
./netmonitor_cpp

# Java (Swing)
javac netmonitor_java.java && java netmonitor_java

# C# (WPF)
dotnet run

# Go (консоль)
go run netmonitor_go.go

# Rust (консоль)
cargo build --release && ./target/release/netmonitor_rs

# JavaScript (Node.js)
npm install blessed blessed-contrib
node netmonitor_js.js

📦 Содержимое репозитория
Файл	Язык	Особенности
netmonitor_python.py	Python	Tkinter + Matplotlib, реальные данные через psutil, сохранение CSV
netmonitor_cpp.cpp	C++	Qt Charts, считывание через системные вызовы (или симуляция)
netmonitor_java.java	Java	Swing, рисованный график, таймер
netmonitor_cs.cs	C#	WPF + LiveCharts, реальные данные через System.Net.NetworkInformation
netmonitor_go.go	Go	консоль, ASCII-график, цветной вывод, симуляция
netmonitor_rs.rs	Rust	консоль с termion, цветной график, симуляция
netmonitor_js.js	JavaScript	Node.js + blessed-contrib, интерактивный консольный интерфейс
🔮 Расширенные функции
Экспорт в CSV для анализа в Excel.
